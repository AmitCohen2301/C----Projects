Information:
This project finds path in a maze between two points (start and target).
If there is no path it will show a message.
The project use OpenGL and base on Bidirectional Search.
The path is found by Bidirectional Search.

Instructions:
In order to find the path u need to right click on the window and press "Run BFS".
In order to restart the maze u need to right click on the window and press "Restart Maze".

Colors:
The space will be displayed in color white.
The wall will be displayed in color black.
The start will be displayed in color blue.
The target will be displayed in color red.
The grey (cells to check) will be displayed in color orange.
The black (cells that checked) will be displayed in color green.
The path will be displayed in color magenta.
