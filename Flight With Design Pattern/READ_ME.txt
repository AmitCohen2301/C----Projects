Information:
This project manage a flight with crew, customers, planes, seats and airport. 
There is a menu with several options to select.

The project use several design patterns:
1. Singleton - only one flight can be produced.
2. Proxy - using an administrative class that does not know the implementation.
3. Command - Keeping all the history of the commands selected by the user (final commands, not switching between menus).

The project also use linked list, vector and string.
